## Sobre

O Quiz Havaianas é um jogo de perguntas e respostas sobre a marca Havaianas. O quiz possui 10 perguntas, cada uma com um tempo limite de 20 segundos para resposta.

Durante o jogo, o usuário recebe feedback visual e sonoro ao responder, o tempo é exibido em tempo real e o fundo da página muda conforme a pergunta atual. Ao final, é apresentada a pontuação total e a lista das questões respondidas incorretamente.

## Primeiros Passos 

O projeto pode ser executado localmente de forma simples, sem necessidade de instalação de bibliotecas ou servidores.

Basta acessar o arquivo principal pelo navegador para iniciar o quiz.

### Pré-requisitos

Para utilizar o projeto, é necessário apenas:

- Um navegador web atualizado (Google Chrome, Microsoft Edge, Firefox, entre outros)

- Um editor de código para visualização ou edição dos arquivos (opcional)

- Manter a estrutura de pastas do projeto organizada

### Instalação

1. Faça o download ou clone este repositório

2. Verifique se a estrutura de arquivos está correta:

- `index.html`

- `style.css`

- `script.js`

- pasta `img/` contendo as imagens

- pasta `sounds/` contendo os arquivos de áudio

3. Abra o arquivo index.html em um navegador

Após isso, o quiz estará pronto para uso.

## Como Usar

1. Abra o projeto no navegador

1. Clique no botão Iniciar

1. Leia a pergunta exibida na tela

1. Selecione uma das alternativas antes que o tempo termine

1. Clique em Próxima para avançar

1. Ao final do quiz, visualize sua pontuação e as questões erradas

1. Utilize o botão Jogar de novo para reiniciar o jogo
